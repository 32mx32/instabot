## 🚀 Быстрый старт

### Установка локально

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/32mx32/instabot.git
cd instabot
```

2. **Создайте виртуальное окружение:**
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

4. **Настройте переменные окружения:**
```bash
cp .env.example .env
nano .env  # или любой другой редактор
```

Заполните файл `.env`:
```env
TELEGRAM_BOT_TOKEN=your_bot_token_from_@BotFather
APIFY_API_KEY=your_apify_api_key
DELETE_AFTER_SEND=true
```

5. **Запустите бота:**
```bash
python main.py
```

## 🐳 Docker развертывание

### Локально с Docker

```bash
# Создайте .env файл с вашими токенами
cp .env.example .env

# Запустите бота
docker-compose up -d

# Просмотр логов
docker-compose logs -f

# Остановка
docker-compose down
```

### На сервере (автоматическая установка)

```bash
# Скачайте и запустите скрипт установки
curl -O https://raw.githubusercontent.com/32mx32/instabot/main/install.sh
chmod +x install.sh
./install.sh
```

Скрипт автоматически:
- ✅ Проверит и установит Docker/Docker Compose
- ✅ Клонирует репозиторий
- ✅ Настроит .env файл (попросит ввести токены)
- ✅ Запустит бота в контейнере

### Обновление на сервере

Для быстрого обновления до последней версии:
```bash
./quick-update.sh
```

## 📖 Использование

### Команды бота

- `/start` - Начать работу с ботом и посмотреть возможности
- `/help` - Подробная справка по использованию
- `/stats` - Статистика работы бота

### Как скачать контент

1. Откройте Instagram
2. Найдите нужный Reels или пост
3. Нажмите **Поделиться** → **Копировать ссылку**
4. Отправьте ссылку боту в Telegram

Бот поддерживает следующие типы ссылок:
```
https://instagram.com/reel/ABC123...
https://instagram.com/p/ABC123...
https://instagram.com/tv/ABC123...
```

### Дополнительные возможности

**Комментарии к постам:**
- Если у поста есть комментарий, появится кнопка "📄 Показать комментарий"
- Нажмите на нее, чтобы увидеть текст в всплывающем окне

**Повторная попытка:**
- Если скачивание не удалось, нажмите кнопку "🔄 Повторить"
- Бот сделает еще 3 попытки

## 🔧 Конфигурация

### Переменные окружения (.env)

| Переменная | Описание | Обязательна | Значение по умолчанию |
|------------|----------|-------------|----------------------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от [@BotFather](https://t.me/BotFather) | ✅ Да | - |
| `APIFY_API_KEY` | API ключ от [Apify](https://console.apify.com/settings/integrations) | ✅ Да | - |
| `DELETE_AFTER_SEND` | Удалять файлы после отправки | ❌ Нет | `true` |

### Настройки в коде

В `main.py` можно изменить:

```python
MAX_REQUESTS_PER_MINUTE = 5  # Лимит запросов на пользователя
MAX_CACHE_SIZE = 1000        # Размер кэша комментариев/URL
```

| Ограничение | Значение |
|-------------|----------|
| Rate limit | 5 запросов/минуту на пользователя |
| Максимум попыток | 3 автоматических + ручные повторы |

- ❌ Instagram Stories (временные истории)

- ❌ Защищенный контент

## 🐛 Troubleshooting

### Бот не отвечает
1. Проверьте, что бот запущен: `docker-compose ps` или `ps aux | grep python`
2. Проверьте логи: `docker-compose logs` или `tail -f bot.log`
3. Проверьте токены в `.env`

### Ошибка "Rate limit exceeded"
- Подождите 1 минуту и попробуйте снова
- Это защита от спама (максимум 5 запросов/минуту)

### Не скачивается контент
1. Проверьте, что ссылка правильная
2. Убедитесь, что аккаунт не приватный
3. Попробуйте кнопку "🔄 Повторить"
4. Проверьте Apify API ключ

### Apify ошибка "Please wait a few minutes"
- Instagram временно заблокировал IP Apify
- Подождите 5-10 минут и попробуйте снова
- Используйте кнопку "🔄 Повторить"
